# Figured Animations — Фикс дрожания ног на элитрах

Исправление бага для аватара [Figured Animations](https://ko-fi.com/s/4b72daaa14)
для мода [Figura](https://modrinth.com/mod/figura) на Minecraft.

⚠️ **Важное предупреждение**

Этот фикс сделан для **старой версии** Figured Animations.  
Автор сейчас работает над полной переделкой (Version 2.0), поэтому фикс может скоро стать неактуальным.

## Баг

При полёте на элитрах ноги персонажа заметно тряслись/дёргались.

## Причина

В файле `EZAnims_modified.lua` (строки 578–587) условие `ob.walking.active` содержало две проблемы:

1. Состояние `gliding` не было исключено из условия ходьбы, из-за чего
   анимация ходьбы могла активироваться прямо во время полёта.
2. Присутствовала строка-фолбэк:

```lua
   or (ob.elytra.active and next(ob.elytra.list) == nil)
```

   Она заставляла анимацию ходьбы включаться каждый раз, когда элитра активна,
   но список анимаций для неё пуст — а это именно то, что происходит при обычном полёте.
   Автор, судя по всему, добавил это как подстраховку, но вышло наоборот.

## Фикс

- Добавлено `and not gliding` в условие ходьбы.
- Строка-фолбэк для элитры убрана полностью.

Оба изменения чистые — никаких костылей, только правка сломанной логики в корне.

## Файлы

Фикс применяется к обоим вариантам аватара (Default/Steve и Slim).
В обоих случаях файл называется `EZAnims_modified.lua` — суффикс в названиях
файлов этого репозитория добавлен просто для различия между ними.

## Кредиты

Оригинальный аватар создан [chmonyasik](https://ko-fi.com/Y8Y5GI5G8/shop).
В этом репозитории находится только исправленный скрипт, не весь аватар.
Как использовать: замените оригинальный `EZAnims_modified.lua` в папке вашего
аватара Figura на файл из этого репо. ("/figura/avatars/Figured_Animations_Default/libs/EZAnims_modified.lua" или "/figura/avatars/Figured_Animations_Slim/libs/EZAnims_modified.lua")
